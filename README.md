Comprehensive computer vision exercises covering image processing fundamentals, organized by lesson with conda environment setup.

---

## Topics Covered

- Image loading and manipulation
- Color spaces and transformations
- Edge detection and filtering
- Feature detection
- Object recognition

---

## Getting Started

```bash
conda env create -f environment.yaml
conda activate cv-nd
jupyter notebook
```
